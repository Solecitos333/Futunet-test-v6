## 2024-05-05 - Missing ARIA Labels on Dynamically Generated Buttons\n**Learning:** Dynamically generated HTML components in JavaScript (e.g., cart and product quantity controls using template literals) lack static analysis coverage for accessibility; ensure `aria-label` attributes are explicitly added to icon-only buttons like `-` and `+` during dynamic generation.\n**Action:** When adding or modifying template strings that output interactive elements (like icon-only buttons), always include explicit `aria-label` attributes.
